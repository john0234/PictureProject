About Me:

I am a recent grad from the University of St. Thomas. I graduated with a BA in Computer Science. I love being outdoors and spending time with friends. This website was made as a way to share my photography to friends and family, but I switched it to also include a little bio about me.

Projects

  • UST QuickAid: This was our Minnehack 2018 project. Our prompt was to create a product that would reduce the cost and occurrence of sports injuries. Our product was a chatbot webapp that would ask yes or no questions to diagnose common sports injuries. We also implemented a chart of the human body to click so that the bot would start asking about that specific body part.
  • My Website (This Webpage): I first started making this separate, but decided to add it on to my photo web page. I am currently hosting it on AWS and I used Bootstrap and JQuery to make it.
  • Stockiratio (Capstone Project): This was our capstone project where our class was split into teams and we worked using the Agile developing method. I worked on the Android team and the Database team.

Classes

  • CISC 131 (Intro to Computer Science): Intro to programming. We learned basic concepts using Python and C. Starter class for Computer Science majors.
  • CISC 230 (Object-Oriented Programming): Class that uses projects to show and teach Object-Oriented Programming methods.
  • CISC 231 (Data-Structures using Object-Oriented Programming): This class taught us about the different methods of storing data and using data structures using Object-Oriented Programming.
  • CISC 210 (Information Security): This class taught us about the most common security mistakes and how to avoid them in our programs.
  • CISC 490 (Software Engineering): This is the Capstone course where we use all of the information we have learned throughout our years to build a team and make a product of our professor's choice.
  • CISC 340 (Computer Architecture): In this class, we went over all the parts of a cpu and looked at innovations in the computing world. The biggest project in this class was creating a pipelined cpu simulator.
  • CISC 310 (Operating Systems): We learned about many things an operating system is in charge of, as well as how they are implemented. This included scheduling, learning about processes, learning about threads and much more.
  • CISC 375 (Web Development): In this class we learned how to use HTML, CSS and Javascript to make a basic webpage.
  • CISC 450 (Database Design): In this class we learned how to design databases to reduce stored redundant data. We also learned how to use mySQL.
  • STAT 360 (Advanced Statistical Software): This class is based around using R to conduct statistical analysis.
